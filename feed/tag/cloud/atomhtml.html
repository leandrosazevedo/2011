Cloud9IDE is one of those new Web IDE, based on Open Web technologies.</p> <p><img src="http://sudweb.fr/blog/public/conferences/cloud9IDE.jpg" alt="cloud9 IDE" style="float: left; margin-top: 0; margin-right: 1em; margin-bottom: 1em; margin-left: 0; " title="cloud9 IDE" />With everything moving to the Cloud, it is now time to have software development join in. JavaScript applications run online for a purpose, so why shouldn&rsquo;t you develop your application online too? So, to do this we are building Cloud9 IDE as a SaaS service with an open source foundation.</p> <p>But what are the advantages of the cloud? In this talk we will go into the advantages that&nbsp; the cloud infrastructure offers to developers and how they can work faster.</p> <p>A major aspect of developing in the cloud is the social aspect. We will explain the future of collaboration when moving development to the cloud. However, there are some challenges to collaboration. We will explain the subtleties of collaboration tooling and the challenges it brings.</p> <p>Why for example did Google Wave get so many complaints, leading to its end. We will explain why collaboration has no single method and use a timeslider/ track changes comparison for purpose of explaining the subtlety.<br />Then we will go deeper into the browser. Advanced widgets need to innovate the browser platform to compete with flash/ silverlight/ java: HTML5. We will talk about HTML5 more in detail.</p> <p>Then we will highlight how easy it is to develop software in the Cloud using Cloud9 IDE, and we will show how to extend the IDE. Ajax.org believes in the open web. Opening up technologies to stand on each other&rsquo;s shoulders, whilst creating healthy businesses. One of the key features of a great IDE is a great code editor. For this part Ajax.org developed Ace, which has recently been adopted by Mozilla as the new Bespin and future code editor for Firefox. The combined power of Mozilla and Ajax.org now makes Ace the best, and most universal code editor on the web.
